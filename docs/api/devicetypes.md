# `/api/devicetypes`

!!! warning

    The api is still under active development. The documentation might not reflect the current implementation. The documentation represents the current end goal for the v1.0 release. Feel free to leave feedback in the site_manager_server repo.

=== "GET"

    **Description:**

    Returns a list of all device types.

    **Path:**

    ```HTTP
    GET /api/v1/devicetypes
    ```

    **Query Parameters:**

    | Name | Type | Description |
    |------|------|---------------|
    | `manufacturerId` | `string | string[]` | (optional) Retrieves only devices from this manufacturer(s) |

    **Request Body:**

    None.

    **Status Codes:**

    | Code | Status |
    |------|------------|
    | `200` | Success |
    | `500` | Internal server error |

    **Response Body:**

    === "Example"
        ```json
        --8<-- "json_examples/GET_devicetypes_res.json"
        ```

    === "Schema"
        ```json
        {}
        ```

    | Name                    | Type                         | Description                                   | Notes |
    |-------------            |---                           |---------------                                | ------- |
    | `id`                    | `string`                     | id of the device type                         |
    | `name`                  | `string`                     | human readable name of the device type        |
    | `manufacturerId`        | `string`                     | id of the manufacturer of the device          |
    | `manufacturerName`      | `string`                     | full name of the manufacturer                 |
    | `manufacturerNameShort` | `string`                     | colloquial name of the manufacturer           |
    | `orderCode`             |`string | null`               | order code as used by the manufacturer        |
    | `orderNumber`           |`string | null`               | order number as used by the manufacturer      |
    | `climateLoad`           | `number | null`              | maximum heat generated by the device in watts |
    | `portTypes`             |                              |                                               |
    | `portTypeId`            | `string`                     | UUIDv4 of the port type                       |
    | `portTypeName`          | `string`                     | name of the port as labeled on the device     |
    | `portTypeDirection`     | `"IN" | "OUT" | "BIDI"`      | direction of the port type                    |
    | `connectorId`           | `string`                     | id of the connector                           |
    | `connectorName`         | `string`                     | human readable name of the connector          |
    | `connectorGender`       | `"MALE" | "FEMALE" | "HERMA“`| gender of the connector                       |
    | `connectorIcon`         | `string`                     | name of the associated icon                   | every icon is available as svg|
    | `compatibleSignalTypes` | `string[]`                   | ids of the compatible signal types            |
    | `moduleSlotTypes`       |                              |                                               |
    | `moduleSlotTypeId`      | `string`                     | UUIDv4 of the module slot type                |
    | `moduleSlotTypeName`    | `string`                     | name of the slot as labeled on the device     |

=== "POST"

    **Description:**

    Creates a new device type.

    **Path:**

    ```HTTP
    POST /api/v1/devicetypes
    ```

    **Query Parameters:**

    None.

    **Request Body:**

    === "Example"
        ```json
        --8<-- "json_examples/POST_devicetypes_req.json"
        ```

    === "Schema"
        ```json
        {}
        ```

    | Name                              | Type                    | Description                                     | Notes |
    |---------                          |---                      |---------------                                  | ------- |
    | `id`                              | `string`                | id of the device type                           |
    | `name`                            | `string`.               | name of the device type                         |
    | `manufacturerId`                  | `string`                | id of the manufacturer of the device type       |
    | `orderCode?`                      | `string | null`         | order code as used by the manufacturer          |
    | `orderNumber?`                    | `string | null`         | order number as used by the manufacturer        |
    | `climateLoad?`                    | `number | null          | maximum heat generated by the device in watts   |
    | `portTypes`                       |                         |                                                 |
    | `portTypeName`                    | `string`                | name of the port as labeled on the device       |
    | `portTypeDirection`               | `"IN" | "OUT" | "BIDI"` | direction of the port type                      |
    | `connectorId`                     | `string`                | id of the connector                             |
    | `compatibleSignalTypes`           | `string[]`              | ids of the compatible signal types              |
    | `moduleSlotTypes`                 |                         |                                                 |
    | `moduleSlotTypeName`              | `string`                | name of the slot as labeled on the device       |
    | `compatibleModuleSlotFormFactors` | `string[]`              | ids of module form factors that fit in the slot |
    | `compatibleModuleTypes`           | `string[]`              | module types that are specifically whitelisted by the manufacturer
    | `incompatibleModuleTypes`         | `string[]`              | module types that are specifically blacklisted by the manufacturer

    |------|------------|
